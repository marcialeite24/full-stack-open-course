# Part 1 - React Fundamentals

First steps with React. This part is about understanding how React thinks — components, props, state and how re-renders work. I also revisited some JavaScript features that are essential when working with React.

## Projects

### [Course Info](courseinfo/)

A static component breakdown of a course page. Focused on how to split a UI into components and pass data down through props.

### [Unicafe](unicafe/)

A feedback collector app with live statistics. The main focus here was learning how to manage state with `useState` and how React re-renders work when state changes.

### [Anecdotes](anecdotes/)

A random anecdote viewer where users can vote for their favourites. Introduced arrays in state and how to update them immutably.

## Key concepts

- Functional components and JSX
- Props for passing data between components
- `useState` for local component state
- Event handlers and how they connect to state updates
- Conditional rendering
- Immutable state updates with arrays
